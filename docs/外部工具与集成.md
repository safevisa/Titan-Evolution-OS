# 外部工具与集成（Agent 现实世界执行层）

本页说明 Titan Evolution OS 如何将 **聊天工具、自媒体、邮件、CRM** 等能力统一进「能力目录」，并在 **租户策略 + 服务端配置 + 用户授权连接** 满足后由 Agent 调用。

## 架构概览

1. **能力目录**（`backend/app/integrations/catalog.py`）  
   每个能力有稳定 `id`（如 `slack_post_message`）、`category`、`auth_mode`、`status`（`live` / `stub`），以及可选的 `connection_provider_any`（租户需在 `integration_connections` 中完成对应连接）。

2. **租户连接**（表 `integration_connections`）  
   - 按 `tenant_id` + `provider` 唯一存储一条连接。  
   - `secret_enc`：Fernet 加密的 JSON（Webhook URL 或 OAuth access/refresh token），**不落明文**。  
   - `meta`：非敏感展示信息（如 Slack team 名、LinkedIn `linkedin_person_urn`）。

3. **加密密钥**（环境变量 `TITAN_INTEGRATIONS_FERNET_KEY`）  
   生成方式：`python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"`  
   未配置时无法保存 Webhook / OAuth 令牌（接口返回 503）。

4. **OAuth 公网回调**（`TITAN_API_PUBLIC_BASE_URL`）  
   必须为 HTTPS 公网 API 根（无尾斜杠），与各平台开发者后台填写的 redirect URI 一致，例如：  
   `{TITAN_API_PUBLIC_BASE_URL}/api/v1/integrations/oauth/slack/callback`

5. **租户策略**（`Tenant.config["integration_grants"]["enabled_capability_ids"]`）  
   - **未配置**：策略上允许目录内能力（执行仍受连接/env 约束）。  
   - **已配置白名单**：仅列表中的 `id` 可执行。  
   - **stub 能力**：仍需显式勾选；当前未实现的 stub 会返回 `stub_not_implemented`。

6. **执行路由**（`app/integrations/executor.py` + `capability_invoke.py`）  
   `execute_capability(...)`：校验白名单、连接、服务端 Key，再调用 HTTP 或 `tool_registry`。

7. **Agent 衔接**  
   `GenericRoleAgent` 支持任务 `input.invoke_capability` + `invoke_capability_params`，结果写入 `output.capability_result`。

## HTTP API（节选）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/v1/integrations/capabilities?tenant_id=` | 目录 + `user_connection_active` / `oauth_app_configured` / `can_execute_now` 等 |
| PATCH | `/api/v1/integrations/tenants/{tenant_id}/grants` | 设置 `enabled_capability_ids` |
| GET | `/api/v1/integrations/tenants/{tenant_id}/connections` | 已连接 provider 列表（无密钥） |
| POST | `/api/v1/integrations/tenants/{tenant_id}/connections/webhook` | Body: `provider` + `webhook_url`（Discord / Slack Incoming / 飞书 / 企业微信） |
| POST | `/api/v1/integrations/tenants/{tenant_id}/connections/credential` | Body: `provider`（`telegram_bot` / `whatsapp_cloud` / `wechat_mp_credentials`）+ `payload` |
| DELETE | `/api/v1/integrations/tenants/{tenant_id}/connections/{provider}` | 删除连接 |
| GET | `/api/v1/integrations/oauth/slack/start?tenant_id=` | 302 → Slack 授权页 |
| GET | `/api/v1/integrations/oauth/slack/callback` | Slack OAuth 回调，写入 `slack_oauth` |
| GET | `/api/v1/integrations/oauth/twitter/start?tenant_id=` | X OAuth 2 + PKCE |
| GET | `/api/v1/integrations/oauth/twitter/callback` | 写入 `twitter_oauth` |
| GET | `/api/v1/integrations/oauth/linkedin/start?tenant_id=` | LinkedIn OAuth |
| GET | `/api/v1/integrations/oauth/linkedin/callback` | 写入 `linkedin_oauth` + `linkedin_person_urn` |
| GET | `/api/v1/integrations/oauth/facebook/start?tenant_id=` | Meta：Facebook Page + 可选 Instagram 商业号 |
| GET | `/api/v1/integrations/oauth/facebook/callback` | 写入 `facebook_graph_oauth` |
| GET | `/api/v1/integrations/oauth/weibo/start?tenant_id=` | 新浪微博 OAuth |
| GET | `/api/v1/integrations/oauth/weibo/callback` | 写入 `weibo_oauth` |
| GET | `/api/v1/integrations/oauth/reddit/start?tenant_id=` | Reddit OAuth |
| GET | `/api/v1/integrations/oauth/reddit/callback` | 写入 `reddit_oauth` |
| GET | `/api/v1/integrations/oauth/google-youtube/start?tenant_id=` | Google OAuth（YouTube Data） |
| GET | `/api/v1/integrations/oauth/google-youtube/callback` | 写入 `google_youtube_oauth` + `youtube_channel_id` |

## 服务端环境变量（平台应用）

| 变量 | 用途 |
|------|------|
| `TITAN_INTEGRATIONS_FERNET_KEY` | 加密存储连接密钥（必填） |
| `TITAN_API_PUBLIC_BASE_URL` | OAuth 回调公网根 URL（OAuth 必填） |
| `SLACK_CLIENT_ID` / `SLACK_CLIENT_SECRET` | Slack OAuth |
| `TWITTER_CLIENT_ID` / `TWITTER_CLIENT_SECRET` | X OAuth 2 |
| `LINKEDIN_CLIENT_ID` / `LINKEDIN_CLIENT_SECRET` | LinkedIn OAuth |
| `FACEBOOK_APP_ID` / `FACEBOOK_APP_SECRET` | Meta Facebook / Instagram（Page + IG） |
| `WEIBO_APP_KEY` / `WEIBO_APP_SECRET` | 新浪微博 OAuth |
| `REDDIT_CLIENT_ID` / `REDDIT_CLIENT_SECRET` | Reddit OAuth |
| `GOOGLE_OAUTH_CLIENT_ID` / `GOOGLE_OAUTH_CLIENT_SECRET` | YouTube Data API（与 `GOOGLE_API_KEY` 分离） |
| `APOLLO_API_KEY` / `RESEND_API_KEY` | 已有 live 能力 |

## 调用参数（invoke_capability_params）

| 能力 id | 说明 |
|---------|------|
| `discord_post_message` | `content` 或 `text` |
| `slack_post_message` | Incoming Webhook：`text`；OAuth 机器人：`channel`（频道 ID 或 `#name`）+ `text` |
| `feishu_lark_notify` | `text` |
| `wechat_work_notify` | `markdown` 或 `text` |
| `twitter_x_post` | `text`（≤280 字符由服务端截断） |
| `linkedin_share` | `text` |
| `telegram_send_message` | `chat_id` + `text`（或 `message` / `content`） |
| `whatsapp_cloud_text` | `to`（E.164，可带 +）+ `text` |
| `facebook_page_feed_post` | `message` / `text`；可选 `link` |
| `instagram_publish_image` | `image_url` + `caption`（需 OAuth 后 meta 含 `instagram_business_account_id`） |
| `weibo_status_post` | `status` / `text` |
| `reddit_submit_post` | `subreddit`、`title`、`kind`（`self` \| `link`）；`self` 用 `text`，`link` 用 `url` |
| `youtube_top_comment` | `video_id` + `text` / `comment` |
| `wechat_mp_template_message` | `touser`（openid）、`template_id`、`template_data`（对象或 JSON 字符串，键为模板字段） |

### 占位能力（中/美主流自媒体无通用自助 API）

目录中 `*_stub` 能力（抖音、小红书、B 站、知乎、视频号、TikTok、Threads、Pinterest、Snapchat 等）用于产品与租户白名单规划；需在 `enabled_capability_ids` 中显式勾选才会进入策略允许集，执行时仍返回 `stub_not_implemented`，待官方或合作接口落地后替换为 `live`。

## 数据库迁移

新增表：`integration_connections`（Alembic revision `006_intconn`）。部署请执行：`alembic upgrade head`。

## 多语言

目录字段 `display_name` / `description` 为中文默认；前端 i18n 建议以能力 `id` 为 key，避免在 API 中堆叠多语言字符串。
