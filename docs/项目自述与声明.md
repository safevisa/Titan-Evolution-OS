# Titan Evolution OS — 项目自述与声明

**文档性质**：本文件为项目方对 Titan Evolution OS（以下简称「本软件」）的**自我说明与使用边界声明**，便于使用者、合作方与审计方理解产品定位与责任范围。  
**版本**：与仓库 `main` 分支同步维护；具体以 GitHub 仓库为准。

---

## 1. 产品是什么

Titan Evolution OS 是一套面向企业与团队的 **「数字员工 + 协同任务流水线」** 软件形态，典型能力包括：

- **多角色 Agent（数字员工）**：按岗位（如研究员、外联、交付、经理等）执行单步任务；
- **协同流水线（goal pipeline）**：按行业插件中的工作流 DAG 串联多角色完成端到端目标；
- **缺口闭环（manager skill closure）**：当流水线中部分阶段因缺岗被跳过时，由管理层任务评估并自动拆解后续补课子任务；
- **技能库（SkillDoc）与记忆层**：将高质量执行沉淀为可检索技能文档，并结合向量记忆等能力支撑复用；
- **绩效与进化相关数据**：通过 `PerformanceLog` 等结构记录执行情况，支撑 KPI、A/B 与后续优化（以实际实现为准）。

本软件强调 **人机协同**：系统辅助编排与执行，**关键业务决策、对外承诺与合规结论仍须由人类负责人确认**。
