## 2. Capability 注册表（TEO-DUAL 新增）

| id | version | category | status | provider / 备注 |
|----|---------|----------|--------|-----------------|
| `context_sync_run` | v1 | context | live | 无 connection；需 context_sync.enabled |

| `computer_use_cancel` | v1 | desktop | live | |

现有能力见 `catalog.py`，勿重复登记。

### 2.1 `context_sync_run` params

```typescript
{
  sources?: ("gmail" | "gcal" | "github")[];
  _idempotency_key?: string;
  _correlation_id?: string;
  _actor?: string;
}
```

### 2.2 `computer_use_submit` params

```typescript
{
  instruction: string;       // required, 1..4000 chars
  max_steps?: number;        // default 30, max 50
  task_id?: string;          // optional UUID link
}
```
