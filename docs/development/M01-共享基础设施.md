## 3. `memory/token_compress.py`

### 3.1 接口

```python
def compress_for_llm(text: str, *, max_chars: int = 12_000) -> str: ...

def compress_html(html: str) -> str: ...
```

### 3.2 规则（按序执行）

1. 若检测 HTML → `compress_html`（markdownify 或简单 strip tags）
2. 折叠 `\n{3,}` → `\n\n`
3. 移除零宽字符、控制字符（保留 `\t\n`）
4. URL query 删除 `utm_*`, `fbclid`
5. 超长截断：保留 head 60% + `\n…[truncated N chars]…\n` + tail 20%

### 3.3 调用点（M01 必须接上）

| 调用点 | 文件 |
|--------|------|
| 工具结果回灌 LLM 前 | `agent_tool_runner.py` 内 tool message content |
| 可选 | `executor.py` 审计 `summarize_capability_result` 之前 |

## 4. `integrations/executor.py` 重构模式

在 `_run_capability_core` 中，在 apollo/resend 之后、invoke_external 之前插入：

```python
from app.context_sync.capability_handlers import try_context_sync_capability
from app.computer_use.capability_handlers import try_computer_use_capability

handled, result = await try_context_sync_capability(...)
if handled:
    return result
handled, result = await try_computer_use_capability(...)
if handled:
    return result
```

各子包 `capability_handlers.py` 返回 `(False, None)` 或 `(True, dict)`。**禁止**在 executor 写超过 10 行业务逻辑。
