### 1.1 必须新增 Celery Beat

当前仅有 worker，**M02 前**必须在主 compose 增加：

```yaml
celery_beat:
  build: ./backend
  entrypoint: ["celery"]
  command: ["-A", "app.workers.celery_app", "beat", "-l", "INFO"]
  environment: # 同 celery_worker
  depends_on:
    redis:
      condition: service_healthy
```

否则 `context_sync.tick` 永不触发。

## 3. 部署顺序（生产变更单）

1. `alembic upgrade head`（009）
2. 部署 backend + celery_worker + **celery_beat**
3. 验证 beat 日志出现 `Scheduler: Sending due task titan.context_sync.tick`
4. 部署 computer-use-runner compose（若启用）
5. 配置 Nginx **不**反代 8090
6. 前端 redeploy（M05）
7. 单租户试点开启 `context_sync.enabled`
